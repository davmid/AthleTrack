AthleTrack – Platforma do Monitorowania Postępów Treningowych
Temat pracy inżynierskiej:
Budowa i wdrożenie webowej platformy do monitorowania postępów treningowych z modułem wizualizacji danych w architekturze full-stack .NET i React.

Opis Projektu
Problematyka pracy inżynierskiej dotyczy braku spersonalizowanych narzędzi umożliwiających efektywne monitorowanie długoterminowych postępów treningowych i kondycyjnych. Celem projektu jest stworzenie nowoczesnej, webowej platformy, która pozwoli użytkownikom na planowanie, rejestrowanie oraz wizualizację wyników i osiągnięć fitness.

Zakres prac:
Backend: ASP.NET Core Web API (C#) – logika biznesowa, autoryzacja JWT, Entity Framework Core.

Frontend: React – dynamiczny i intuicyjny interfejs użytkownika.

Architektura: Full-stack, separacja logiki (REST API) od prezentacji (SPA).
